# Overview

| | A: Deployment Process | B: Configurations/Secret Management |
|:---|:---|:---|
| Maturity 1 - Deployment processes are fully documented | Deployment is automated or done by someone other than the developer. | Configuration files are versioned |
| Maturity 2 - Deployment processes include security verification milestores | Integration of security verification in deployment (e.g. binary static code analysis / AV scan) | Configuration files do not contain unencrypted sensitive configuration information |
| Maturity 3 - Deployment process if fully automated and incorporates automated verification of all critical milestones | All code is digitally signed and signatures are verified at Deployment | Deployment process automatically generates / initializes credentials and authentication keys |
